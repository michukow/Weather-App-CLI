# Weather CLI App (Python)

Simple command-line application that fetches current weather data
based on latitude and longitude using the Open-Meteo public API.

## Features
- user input validation
- public API usage (no API key)
- JSON data saving
- display: temperature, wind direction, wind speed
- drawing a chart: rain and max. temperature for the next 7 days

## Technologies
- Python
- requests, matplotlib
- Open-Meteo API
- Nominatium OpenStreet
