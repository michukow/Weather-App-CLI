# Weather Window App 2.1 (Python)

Simple widnow application that fetches current weather data
based on latitude and longitude using the Open-Meteo public API.

## Features
- user input validation
- public API usage (no API key)
- JSON data saving
- simple window:
	- with longitude & latitude input
	- displaying: date, time, max temperature. wind speed and wind direction
	- drawing simple chart with 7-days forecast

## Technologies
- Python
- requests, matplotlib
- Open-Meteo API
- Nominatium OpenStreet
- CustomTkinter
